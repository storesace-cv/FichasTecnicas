# Ficha técnica com fotografia do prato

A página de ficha técnica passa a destacar a fotografia do prato sempre que o registo traz um ficheiro associado. O bloco de imagem aceita URLs absolutas ou relativas devolvidas pelo backend e aparece ao lado do resumo/atributos, com sobreposição para texto descritivo.
